About this project!

What is it?

This a simple skeleton of an AngularJS web application. And with the help of Grunt and Bower it is possible to use this skeleton as a starting point for frontend development with AngularJS.

What for?

The main idea behind this little project is to learn how to build an application with AngularJS. And as a really nice *side effect* the automation based on Grunt and Bower takes away some of the repetitive and annoying tasks.

Stack

The application

  • AngularJS v1.2.0rc
  • jQuery v2.0
  • Twitter Bootstrap v3.0
  • Modernizr v2.6.2

The build process

  • Grunt v0.4.1
  • Bower v1.2.6
  • Karma v0.10.2

Installation

  1. Install Node.js and NPM
  2. $ npm install -g grunt-cli
  3. $ git clone git@bitbucket.org:nosch/ng-scaffold.git "ng-scaffold"
  4. $ cd ng-scaffold/client/
  5. $ npm install
  6. $ bower install

How to use

Start to develop, to test or to build an application with one of the following grunt tasks:

  • $ grunt dev-server
  • $ grunt release && grunt release-server
  • $ grunt test
